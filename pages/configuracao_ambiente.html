1. Configuração do Ambiente

O passo crucial para o funcionamento correto do sistema como um todo é a configuração do ambiente físico.

Toda a parte de automação de irrigação depende diretamente deste ambiente, porém as funcionalidades de gestão podem ser utilizadas normalmente sem o mesmo.

Os itens necessários para isto são:

  1. Placa Arduino

    A placa Arduino é utilizada para executar um código que é responsável pelo recebimento das informações vindas do dispositivo Android. Estas informações são processadas e liberam um comando de ação para o circuito.

  2. Circuito Irrigare

    O circuito Irrigare é alimentado com energia elétrica pelo Arduino. Realizando as ações que foram solicitadas pelo Arduino, o sistema ativa ou inativa uma válvula solenóide conectada ao circuito.

  3. Válvula Solenóide

    A válvula solenóide utilizada neste sistema foi uma válvula solenóide de diâmetro de passagem de água 3/4" e ativada por corrente elétrica 12VDC.

  4. Roteador Wireless

    O Roteador realiza a conexão entre o dispositivo Android e o Arduino, a comunicação é estabelecida através da rede.

  5. Cabo de Rede RJ45

    O cabo de rede é utilizado para conectar o roteador à placa Arduino.

Após o ambiente físico estar montado, é necessário realizar o upload do sistema para a placa Arduino. Antes de enviar o sistema o endereço IP da placa Arduino deve ser definido de acordo com a faixa de IP da rede LAN.

Com o endereço de IP definido, o usuário deve abrir o aplicativo Android navegar até o item "Configurações", preencher o campo IP com o endereço que foi definido na placa Arduino e clicar no botão "Salvar".

O próximo passo é realizar o cadastro da válvula no aplicativo Android, acessando o item "Cadastro de válvulas". Nesta tela é necessário preencher os campos com o nome da válvula, o campo opcional "Local" e atenção ao campo Identificador Físico. No campo identificador físico é necessário apontar a porta física cuja a válvula está conectada.

Devido às limitações físicas do circuito, podem ser definidas no máximo 5 válvulas no cadastro.

Com estas configurações realizadas o sistema está pronto para ser utilizado.
