9. Gráficos

Nesta funcionalidade o usuário pode gerar gráficos a respeito do cultivo.

Existem cinco tipos de informações diferentes que o sistema pode retornar ao usuário:

  1. Quantidade de insumos gastos X período
    Este gráfico é gerado a partir do somatório das quantidades utilizadas de um ou mais insumos, de acordo com o período especificado pelo usuário.

  2. Culturas que mais gastaram insumos X período
    Este gráfico é gerado a partir do somatório das quantidades de insumo utilizado por cultura, de acordo com o período especificado pelo usuário.

  3. Culturas que mais irrigaram X período
    Este gráfico é gerado a partir do somatório das quantidades de irrigação por válvula associada a cultura, de acordo com o período especificado pelo usuário.

  4. Culturas que mais consumiram em R$ X período
    Este gráfico é gerado a partir do somatório das quantidades de insumo utilizados por culturas, multiplicado pelo valor de compra dos insumos, de acordo com o período especificado pelo usuário.

  5. Culturas mais lucrativas X período
    Este gráfico é gerado a partir do somatório das quantidades de insumo utilizados por culturas, multiplicado pelo valor de compra dos insumos, levando em conta as culturas que menos consumiram, de acordo com o período especificado pelo usuário.

O usuário pode ainda optar por dois formatos diferentes de gráfico:

  1. Formato de pizza

  2. Formato de barras
